A tag 'bdo' é usada para definir a direção de um texto

Essa tag substituí a direção do texto atual e deve obrigatóriamente ter o atributo 'dir'

Css padrão:
bdo {
unicode-bdi: bdi-override; }
Aceita todas tags globais (Uso obrigatório do atributo global DIR)
