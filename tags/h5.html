The tag h5 is used to make the 5th heading of your page;

Can support all global-attributes;

Remember: you need have all other heading that come before the tag h5 and they need to be in the same context (tag);

Default css:

display: block;
font-size: .83em;
margin-top: 1.67em;
margin-bottom: 1.67em;
margin-left: 0;
margin-right: 0;
font-weight: bold;
