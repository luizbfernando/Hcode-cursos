A tag 'dfn' é usada para definir a instância de definição de um termo

A instância de definição normalmente é o primeiro uso de um termo

A tag pai de dfn também deve conter a explicação para o termo

Possui css padrão e aceita todos os atributos globais

ins {
font-style: italic;
}
