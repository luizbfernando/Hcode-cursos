A tag 'hr' é usada para definir uma quebra temática na página

Esta tag é normalmente usada para separar o conteúdo

Possui css padrão e aceita todos os atributos globais

hr {
display: block;

margin-top: 0.5em;

margin-bottom: 0.5em;

margin-left: auto;

border-style: insert;

border-width: 1px;
}
