Tag 'rt' é usada para definir o texto de uma notação em 'ruby'

A tag fornece informações de pronúncia, tradução ou transliteração do Leste Asiático

A tag deve ser sempre usada dentro da tag 'ruby'

Possui css padrão e aceita todos os atributos globais

rt {
line-height: normal;
}
