A tag 'q' é usada para fazer uma citação externa PEQUENA e SIMPLES

Deve, obrigatóriamente, ter o atributo 'cite'

Css padrão:
q {
display: inline;
}
q:before {
content: open-quote;
}
q:after {
content: close-quote;
}
