A tag 'cite' é usada para definir o título de uma obra (livro, filme, música, pintura, etc )

A tag não possui conteúdo e deve ser usada para quebrar linhas, não separarando os parágrafos

Possui css padrão e aceita todos os atributos globais Css padrão:
cite {
font-style: italic;
}
