Tag 'fieldset' define um grupo de campos de formulários

Esta tag define uma caixa ao redor dos formulários

Pode conter uma legenda explicando seu uso

Possui css padrão e aceita todos os atributos globais

fieldset {
display: block;

margin-left: 2px;

margin-right: 2px;

padding-top: 0.35em;

padding-left: 0.75em;

padding-bottom: 0.625em;

padding-left: 0.75em;

padding-right: 0.75em;

border: 2px groove (international value);
}
