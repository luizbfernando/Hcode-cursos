Tag 'dd' define a descrição termo em uma lista com descrição

A tag é usada em conjunto da tag 'dt' e dentro da tag 'dl'

Pode conter quebras de linhas, parágrafos, imagens etc.

Possui css padrão e aceita todos os atributos globais

dd {
display: block;

margin-left: 40px
}
