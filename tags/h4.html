The tag h4 is used to make the 4th heading of your page;

Can support all global-attributes;

Remember: you need have all other heading that come before the tag h4 and they need to be in the same context;

Default css:

display: block;
font-size: 1em;
margin-top: 1.33em;
margin-bottom: 1.33em;
margin-left: 0;
margin-right: 0;
font-weight: bold;
