Tag 'u' é usada para definir uma parte de um texto sublinhado

É usada normalmente para indicar que uma palavra está com erro de ortografia

Evite usar essa tag para não confundir o usuário com um hyperlink

Possui css padrão e aceita todos os atributos globais

u {
text-decoration: underline;
}
