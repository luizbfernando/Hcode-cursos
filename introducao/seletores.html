Os seletores CSS3 servem para selecionar uma tag ou objeto do seu arquivo html para que os estilos sejam adicionados.

Seletor universal

Seleciona o seu documento html inteiro

*{
background: black;
}

Seletor por tipo

Seleciona uma tag em específico

body {
background: black;
}

Seletor por classe

Seleciona uma classe em específico

.fundo-preto {
background: black;
}

Seletor por ID

Seleciona um id em específico (que não irá se repetir)

#nome {
background: black;
}

Seletor por Pseudos

Aplica o atributo quando o cursor do mouse passa por cima de algum elemento

:hover {
background: black;
}

Sintaxe

Header, section, footer -> Grupo de seletores
{ background: black; } -> Bloco de declarações (Propiedade da declaração, separador, valor da declaração e fim da declaração)
