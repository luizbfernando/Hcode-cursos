Modelo de Caixa

Todos os elementos HTML podem ser considerados caixas. No CSS, o termo "modelo de caixa" é usado quando se fala em desgin e layout


O modelo de caixa CSS é essencialmente uma caixa que envolve todos os elementos HTML. Consite em: margens, bordas, preenchimento, e o conteúdo.

O border-box faz com que a largura seja fixa, nenhuma medida além dela pode mudar o tamanho do elemento

A ordem é Top, right, bottom e left
