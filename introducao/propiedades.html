Valores Globais

Os valores globais podem ser utilizados em todas as propiedades CSS a menos que tenha alguma regra específica como exceção

Inital

Este valor irá forçar a reinicialização do valor padrão dela

Inherit

Este valor irá atribuir a herança explícita para a propiedade aplicada. Assim caso o elemento pai possua a mesma propiedade com algum valor, este será aplicado para o elemento em questão

Unset

Desabilita a propiedade em questão

Revert

Este valor tem a mesma função do valor inital porém é aplicado somente no navegar safari.
