Combinadores de Seletores

Combinador Descendente

form input { color: blue;}

Combina com o espaço entre os seletores o css do seletor pai irá ser aplicado também ao seletor filho

Combinador Filho direto

form > input { color : blue;}

O estilo css será aplicado somente se o segundo seletor for um filho direto do primeiro seletor.

Combinador Irmão Adjacente

form + div { color : blue; }

Irá aplicar o css apenas na div que for a irmã mais próxima e após a tag form.

Combinador Irmão geral

form ~ div { color : blue; }

Todas as divs quem estiverem logo após a tag form, terão a cor azul
