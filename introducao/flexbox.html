Flex Box Module

O Flexbox é capaz de organizar os elementos de uma interface de modo unidimensional.

É possível definir a organização dos elementos como linha ou coluna e usar recursos para alinhar e distribuir após isso.

Eixos do flexbox

Ao definir a dimensão (linha ou coluna) por meio da propiedade flex-direction, este eixo será tratado como principal e o outro como eixo transversal
