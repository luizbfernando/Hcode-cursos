Layout em Grade

A especificação CSS Grid Layout permite a criação de um layout é possível criar layout semelhante ao que era feito com tabelas, porém diferentemente das tabelas, os elementos filhos podem se sobrepor a outros conteúdos, dando aos elementos HTML o poder de ser alocado em qualquer parte do layout.

Assim como o módulo FlexBox, o módulo Grid Layout também irá trabalhar com um container que funcionará como um wrapper ou embrulho dos elementos internos denominado grid items.

Por que Grid?

Quando trabalhamos com layout podemos combinar o uso de grid com outras tecnologias, surge a questão. Por que utilizar o Grid Layout? Os sites construídos com flexbox tem flexibilidade unidimensional que permite que diferentes elementos do site sejam redimensionados dependendo do tamanho da tela. No entanto há mais no design de um site do que simplesmente posicionar elementos de uma determinada maneira

Com Grid, temos a liberdade de reproduzir layout mais livremente de acordo com os conceitos que o desiger produziu, manter esses conceitos, não era uma tarefa fácil, mesmo usando flexbox, com Grid fica mais fácil criar layout multidimensionais.

Grid Container

O elemento que receberá a propiedade display: grid é chamado Grid Container e podemos dizer que será o elemento pai de todos os itens de grade

Grid Lines

Qualquer linha vertical ou horizontal de uma grade que faz as divisões entre linhas e colunas são Linhas de Grade ou Grid lines

Grid Cells

Cada espaço de uma grade ou Grid Cell, também conhecida como unidade de grade

Grid tracks

O espaço entre duas linhas adjacentes, horizontais ou verticais, são chamados de Faixa de Grade ou em inglês Grid Track

Grid Area

O espaço total gerado por quatro linhas de grade, formando um grupo de céculas chamado de Área de Grade ou Grid Area

Gutters

Gutters são os espaços entre as linhas ou colunas de grade. Como que calhas podem fazer uma separação entre esses elementos da nossa grade
