Propriedade box-decoration-break

Esta propriedade CSS define como plano de fundo, preenchimento, borda, imagem de borda, borda, sombra da caixa, margem e caminho do clipe de um elemento quando são aplicados quando a caixa do elemento é quebrada.

Valores

Valores possíveis:none, h-offset, v-offset, blur, spread, color, inset, initial e inherit

Valor Padrão:none

Valor Herdado:não

Acesse
o hcode
class
