Propriedade overflow-x

Esta propriedade CSS especifica se o conteúdo deve ser cortado, adicionado uma barra de rolagem ou exibir o conteúdo do estouro de um elemento de nível de bloco quando este estoura as bordas laterais.

Valores

Valores possíveis:visible, hidden, scroll, auto, initial, inherit, unset

Valor Padrão:visible

Valor Herdado:não
