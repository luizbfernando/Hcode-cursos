Propiedade border-image

Define uma imagem como a borda ao redor do elemento

Esta propiedade é uma forma abreviada das propiedades border-image-source, border-image-width, border-image-slice, border-image-outset, border-image-repeat.

Valores

Valores possíveis:source slice, width, outset repeat, initial e inherit

Valor Padrão:None 100% 1 0 stretch

Valor Herdado:não
