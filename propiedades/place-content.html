Propriedade place content

Esta propriedade CSS é uma abreviação das propriedades align-content e justify- content

Se o segundo valor não for informado, o primeiro valor será usado para ambos;

Valores

Valores possíveis:normal, strech, center, start, end flex-start, flex-end, self-start, self-end, baseline, first baseline, last baseline safe center, unsafe center, initial, inherit, unset

Valor Padrão:normal

Valor Herdado:não
