Propriedade overflow-wrap

Esta propriedade css é aplicada a elementos de linha, adicionando uma quebra de linha sempre que uma palavra for gerar um estouro na caixa

Valores

Valores possíveis:normal, break-word, anywhere, initial, inherit, unset

Valor Padrão:normal

Valor Herdado:Sim
