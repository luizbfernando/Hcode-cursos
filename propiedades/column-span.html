Propriedade column-span

Esta propiedade CSS define se um elemento deve acupar todas as colunas ou apenas uma coluna do elemento onde está inserido.

Valores

Valores possíveis:none, all, initial, unset e initial

Valor Padrão:none

Valor Herdado:Não
