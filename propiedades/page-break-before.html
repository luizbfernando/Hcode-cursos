Propriedade page-break-before

Esta propiedade CSS define se haverá uma quebra de página, coluna ou região dentro de um determinado elemento começar. Muito útil para impressões

É a exata mesma coisa do break-before(nome antigo) mas com o nome atual

Valores

Valores possíveis: auto, all, always, avoid , avoid-column, avoide-page, avoid-region, column, left, page, recto, region, right, verso, initial, inherit e unset

Valor Padrão:auto

Valor Herdado:Não
