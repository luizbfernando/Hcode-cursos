Propriedade Background-clip

Esta propriedade CSS define até que ponto o plano de fundo deve se estender dentro de um elemento

Valores

Valores possíveis:border-box, padding-box, content-box, initial, inherit e unset

Valor Padrão:border-box

Valor Herdado:não
