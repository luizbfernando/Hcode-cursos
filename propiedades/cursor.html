Propriedade Cursor

Esta propiedade CSS define o cursor do mouse que será exibido quando o mouse passar por determinado elemento.

Valores

Valores possíveis:Auto, cursor-type, initial, inherit, unset

Tipos de cursor:Alias, all-scroll, auto, cell, col-rezise, context-menu, copy, crosshair, default, e-resize, ew-resize, grab, grabbing, help, move, n-resize, ne-resize, nesw-resize, ns-resize, nw-resize, nwse-resize, no-drop, none, not-allowed, pointer, progress, row-resize, s-resize, se-resize, sw-resize, text, url, vertical-text, w-resize, wait, zoom-in e zoom-out.

Valor Padrão:Auto

Valor Herdado:sim
