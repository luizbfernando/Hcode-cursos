Propriedade overflow-block

Disponível apenas no firefox

Esta propriedade css determina como um elemento HTML que possua essa propriedade poderá ser redimensionado pelo usuário

Esta propriedade irá direcionar para as propriedades overflow-y ou overflow-x dependendo do modo de escrita do documento

Valores

Valores possíveis:None, both, horizontal, vertical, block, inline, initial, inherit, unset

Valor Padrão:none

Valor Herdado:não
