Propriedade place-items

Esta propriedade CSS é uma abreviação das propriedades align-items e justify-items

Se o segundo valor não for informado, o primeiro valor será usado para ambos;

Valores

Valores possíveis:legacy, legacy right, auto, normal, strech, center, start, end flex-start, self-start, self-end, baseline, first baseline, last baseline, safe center, unsafe center, initial, inherit, unset

Valor Padrão:auto

Valor Herdado:não
