Propriedade column-rule-width

Esta propiedade CSS define a largura das linhas divisórias entre as lacunas de um elemento dividido em colunas

Valores

Valores possíveis:medium, thin, thick, lengh, initial inherit e unset

Valor Padrão:medium

Valor Herdado:Não
