Propiedade box-sizing

Esta propiedade css define como a largura e altura de um elemento são calculadas. Se irão incluir o preenchimento e bordas ou não;

Ela é uma abreviação das propiedades border-width, border-style e border-color

Valores

Valores possíveis:content-box, border-box, initial, inherit

Valor Padrão:content-box

Valor Herdado:não
