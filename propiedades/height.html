Propiedade Height

Define a altura dos elementos, o que não incluí as margens, bordas e espaçamento.

Permite herança explícita, uma vez definida a altura do elemento pai é possível herdar para o elemento filho.

As propiedades max-height e min-height sobrescrevem a Propiedade height

Valores

Valores possíveis:auto, length, initial, inherit

Valor padrão:não

Valor Herdado:não
