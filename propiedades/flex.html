Propriedade flex

Esta propriedade CSS define o comprimento flexível dos items flexíveis.

Esta propriedade é uma forma abreviada das propriedades flex-grow, flex-shrink e flex-basis.

Valores

Valores possíveis:flex-grow, flex- shrink, flex-basis, initial inherit, unset.

Valor Padrão:0, 1 , auto

Valor Herdado:não
