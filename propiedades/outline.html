Propriedade outline.

Esta propriedade é uma forma abreviada das seguintes propriedades: outline-width; outline-style; outline-color

Esta propriedade CSS define o espaço entre a borda e a linha de contorno do elemento.

Valores

Valores possíveis:outline-width, outline-style, outline-color, initial, inherit e unset

Valor Padrão:medium auto invert

Valor Herdado:não
