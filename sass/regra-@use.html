A regra @use

Essa regra carrega mixins, funções e variáveis de outras folhas de estilo CSS e combina CSS de várias folhas de estilo.
Folhas de estilos carregadas usando o @use são chamadas de módulos. O SAAS também fornece módulos integrados com muitas funções úteis.

A regra @use mais simples é escrita usando o @use "< url >" que carrego o módulo fornecido. Todos os estilos carregados dessa forma serão incluídos exatamente uma vez na saída CSS compilada, não importa quantos estilos sejam carregados

Dica: Como o @use adiciona Namespacesa aos nomes dos membros, é seguro escolher nomes muito simples como $radius ou $width ao escrever uma folha de estilo

Como utilizar

Arquivo corner.scss

$radius: 3px;

@mixin rounded {
border-radius: radius
}

Arquivo style.css

@use "src/corners";
.button {
@include corners.rounded;
padding: 5px + corner.$radius;
}

Arquivo final - style.css

Arquivo style.scss

.button {
border-radius: 3px;
padding: 8px;
}

Utilizando Namespaces

@use "src/corners"; as c
.button {
@include c.rounded;
padding: 5px + c.$radius;
}
