O que é o Sass

É uma linguagem de extensão (dlc) de folhas de estilos de nível profissional mais madura, estável e poderosa.

É compatível com todas as versões do CSS

Tem sido apoiado ativamente por mais de 14 anos e tem sido escolhido pela indústria como a principal extensão do CSS

Por que usar Sass?

Os pré-processadores são linguagens de script que estendem os recursos padrão do CSS

Eles permitem usar a lógica dentro do código CSS, como variáveis, alinhamento, herança, mixins, funções e operações mateméticas

Os pré-processadores CSS facilitam a automatização de tarefas repetitivas, reduzem os números de erros e inchaço do código, criam fragmentos de código reutilizáveis e garantem compatibilidade com versões anteriores
