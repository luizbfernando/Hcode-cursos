Instalando o Sass

Requisitos

Node.js (No windows). Para instalar por linha de comando, digite

npm install -g sass

O sass irá monitorar o diretório src/sass buscando por alterações, e irá compilar o CSS final para a pasta dist/assets/css

O parâmetro --watch

É possivel monitorar um arquivo ou um diretório com a flag --watch. A flag --watch irá avisar toda vez que um diretório monitorado for alterado, e então irá recompilar o arquivo sass

Para ativar o monitoramento de um arquivo sem ter que fazer isso manualmente, basta adicionar o comando --watch: sass -watch input.scss output.css

Monitorando um diretório

É possível monitorar um diretório completamente, e gerar um código correspondente em outro diretório, basta utilizar os dois pointos :

sass --watch src/sass:dist/assets/css
