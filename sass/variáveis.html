O que são variáveis no Sass?

Semnpre que for preciso usar o mesmo valor em vários lugares, é recomendado usar variáveis

As variáveis no Sass são simples: basta atribuir um valor em um nome que começa com $ e, em seguida, pode se referir a este esse com o nome em vez do própio valor

Apesar de sua simplicidade, elas são umas das ferramentas mais úteis no sass

Variáveis do Saas X Variáveis do CSS

O CSS possui variáveis nativamente mas essas são bem diferentes das do Saas

  • Variáveis escritas no Saas são totalmente compiladas, gerendo um código final estático já as variáveis do CSS são mantidas no arquivo final .css
  • Variáveis no Saas sempre terão o mesmo valor, o que não acontece no CSS.
  • Variáveis no Saas são imperativas, ou seja, se você mudar seu valor após sua declaração, o valor anterior permanece o mesmo. Já se isso for feito no CSS, como as variáveis são declarativas, o valor afetará o uso anterior e futuro da variável

Utilizando variáveis

SASS
$variável: 10px;
.class {
value: $variável;
}
CSS
.class {
value: 10px;
}

Hífen e underscore são iguais. Isso significa que $font-size = $font_size
