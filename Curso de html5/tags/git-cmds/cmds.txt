Git add "exemplo-arquvio.txt" = adiciona um arquivo ao seu controle de versão (git add . para adicionar todos de uma vez só);

git status = Mosta as mudanças que foram realizadas e/ou o que precisa ser adicionadas ao controle de versão;

Git commit -m "Exemplo cmd" = Cria uma versão do seu código para ser enviada, uma boa prática é descrever o que foi feito na mensagem (nas aspas duplas);

git push = Envia uma versão do seu código para uma nuvem remota. Precisa ter um link atrelado ao arquivo ou pasta (git remote add origin <exemplodesite.com>);

git reflog = Mostra as versões que foram feitas (As mais recentes ficam no topo);

git reset --hard <seu-codigo-reflog> = Retorna a versão do código atrelada ao código reflog específicado. Muito útil para navegar entre versões do seu código, caso cometa algum erro, por exemplo;

git branch = Mostra as branchs disponíveis ;

git branch <nome-branch> = Cria uma nova branch;

git checkout <nome-branch> = Muda para a branch especificada.

git merge <branch para mergiar com a atual> = Envia/une o conteúdo de uma branch com outra branch.
